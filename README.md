<h2 style="color:#2e7d32;">📌 Project Overview</h2>

<p>
HaritBandhu is a <b style="color:#1565c0;">scalable backend system</b> powering an AI-driven agriculture platform designed for Indian farmers.
It provides intelligent APIs for pest detection, AI chat assistance, weather insights, and government schemes.
</p>

<p>
<b style="color:#6a1b9a;">"Technology ke saath, Kisan aage badhe" 🌱</b>
</p>

---

<h2 style="color:#2e7d32;">🎯 Problem Statement</h2>

<ul>
<li>Lack of real-time pest and disease detection</li>
<li>No unified platform for mandi prices</li>
<li>Limited awareness of government schemes</li>
<li>Language barriers in agri-advisory</li>
<li>Poor data-driven farming decisions</li>
</ul>

---

<h2 style="color:#2e7d32;">✨ Key Features</h2>

</ul>
